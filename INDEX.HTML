PRACTICA 1

La sintaxis basica de CSS

selector{
propiedad:valor;
}
selector:Es el elemento o los elementos que vamos a cambiarle la propiedad
Propiedad:que es lo que deseamos cambiar como el ancho,el color,la posicion
el tamaño,agregamos bordes etc

Tipos de Selectores

Universal:Selecciona todos los elementos y cambia la forma de color
*{
color;red
}
De tipo:los selectores de tipo son los que seleccionamos por elementos
h1{
color;red
}

Clase: Agregar el atributo
class="titulo-h2"
.titulo-h2
solo cambiamos el color de un solo elemento
ID: Para seleccionar un elemento se utiliza el
#id-elemnt{
color:red
}
con este selector se identifica un elemento unico, para poder trabjar de la forma coreecta
no se pueden repetir es incorrecto utilizar un mismo id para muchos elementos
Por atributo: Para utilizar este selector se utiliza
[real]="sii"{
color greed
}

puede ser cualquier atributo
Descendiente:Se aplica a los elementos que estan dentro
de otro elemento (o mas exactamente, un elemento que es descendiente
de otro elemento).Por ejemplo,una lista desordenada tiene una etiqueta con
descendientes
h2 p {
color:red;
}
Pseudo-clases:Especidica un estado especial del elemento seleccionado.Por ejemplo:hover
aplicara un estilo cuando el usuario haga hover sobre el elemento especificado por el
selector.
b:hover{
color:red;
}
